# Version 4.0.1

## Changes

### Removed (breaking)

- Removed the `--body-template` flag from `gmail sendmail` and
  `msgraph sendmail`. `--template` (introduced in 3.6.10 on all five sendmail
  protocols) fully supersedes it: a non-`.eml` file is used as the HTML body,
  with Go `text/template` variable substitution on top. The flag, its
  environment variables, and its config-file key are gone; passing
  `--body-template` now fails with `unknown flag`.

  Migration:

  | Before | After |
  |--------|-------|
  | `--body-template file.html` | `--template file.html` |
  | `MSGRAPHBODYTEMPLATE` | `MSGRAPHTEMPLATE` |
  | `GMAILBODYTEMPLATE` | `GMAILTEMPLATE` |
  | YAML key `body-template` | YAML key `template` |

  Note: `--template` renders the file through `text/template` with
  `missingkey=error`, so a template file containing literal `{{` (e.g. in
  inline JavaScript or CSS) must escape it as `{{"{{"}}`.

### Documentation

- Removed `--body-template` from `docs/protocols/{gmail,msgraph}.md`,
  `docs/config-file.md`, and the msgraph sendmail YAML/env examples.
